# Inventory Management System – Case Study

This repository contains the solution to a multi-part backend case study focused on designing, debugging, and implementing features for a B2B inventory management system.

The work is divided into three parts:
- Part 1: Debugging an existing API
- Part 2: Database schema design
- Part 3: Low-stock alert API implementation
